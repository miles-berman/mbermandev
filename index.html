City Views

City Views is a personal project I have been working on for quite a while. The idea behind this game was to create an experience where the player has very limited choices for movement and interaction in the world, and is forced to just sit and watch the world around them. I wanted to challenge myslef to create the game from scratch, working on all the music, art, animations, and game mechanics myself. It is a project I am very proud of, and demonstrates a lot of my skills, including:

  • Algorythmic Music Generation
  • Animation And Art Skills
  • C# Programming and Unity Development
  • Solo Game Development Skills

SMPL-HLPR

A python "sampler" to change bit depth, sample rate, and pitch of WAV, MP3, & M4A files. The app allows users to load files and manipulate the audio in real-time. Users can adjust the bit depth, sample rate, and pitch of the audio, and apply these changes on the fly.

  • Python Programming
  • Audio File Manipulation
  • UI and App Distribuion
