Part 3: Generations

The children

To start off with a simple example, let's choose two parents at random and create a child from them. This child will get half of its data from each parent in attempt to create a new organism that is a mix of the two.

Updated test

To implement this into our organism class, we can simply use the following code: 


# Python code for a basic children from parents

def child(self, partner):
    child = Organism()
    child.new_brain()
    for i in range(len(self.synapses)):
        if random.random() > 0.5:
            child.synapses.append(self.synapses[i])
        else:
            child.synapses.append(partner.synapses[i])
    return child
